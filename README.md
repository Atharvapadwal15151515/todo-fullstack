# TaskFlow

A modern full-stack Todo application built using React, Spring Boot and PostgreSQL.

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Spring Boot
- Spring Data JPA

### Database
- PostgreSQL

## Features

- Add tasks
- Delete tasks
- Mark tasks complete
- Dashboard statistics
- Search functionality
- Responsive design

## Project Structure

```text
todo-fullstack
├── demo        (Spring Boot Backend)
└── frontend    (React Frontend)
```

## Installation

### Backend

```bash
cd demo
.\mvnw spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
